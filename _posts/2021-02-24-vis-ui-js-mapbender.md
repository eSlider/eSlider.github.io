**vis-ui.js** is a JavaScript web UI generator built for the **Mapbender** geoportal framework during employment at WhereGroup GmbH.

**Repositories**:

- [github.com/eSlider/vis-ui.js](https://github.com/eSlider/vis-ui.js)
- Upstream: [mapbender/vis-ui.js](https://github.com/mapbender/vis-ui.js)

## Context

Mapbender is WhereGroup's open-source geoportal framework ([OSGeo](https://www.osgeo.org/projects/mapbender/)), deployed for German public-sector clients serving **1M+ users**.

vis-ui.js generates consistent UI components across Mapbender modules — reducing bespoke front-end work per client deployment.

## Related work

Built alongside the **[Digitizer](/posts/mapbender-digitizer-module/)** spatial editing module and print/search integrations — core product modules adopted across Mapbender installations.

## Tech stack

JavaScript · Mapbender · WebGIS · Symfony (host framework)
